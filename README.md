Election Assistant – User Guide
Election Assistant is an interactive website that helps users understand the election process in a simple, role-based way. The platform is designed for three main user groups: voters, candidates, and volunteers.

What this website does
The website helps users:

Understand election steps in an easy-to-follow format.

Choose a role and explore the correct process for that role.

Learn about election timelines, booth flow, and result-related stages.

Interact with guided simulations instead of only reading static content.

Who can use it
This platform is built for:

Voters who want to understand registration, booth flow, and result stages.

Candidates who want to understand the election participation process.

Volunteers who want to learn booth duties and closing procedures.

First-time users who need a simple and guided explanation of election activities.

How to use the website
1. Open the website
Launch the Election Assistant website in your browser.

2. Start from the home screen
On the main screen, read the introduction to understand the purpose of the platform.

3. Select your role
Choose the option that best matches you:

Voter

Candidate

Volunteer

The website will guide you based on the selected role.

4. Begin your journey
After choosing a role, the system starts a guided journey. This journey is personalized to the role you selected and shows the relevant election process.

5. Explore interactive sections
Depending on your role, you may see modules such as:

Role-based journey steps

Polling booth simulation

Volunteer booth simulation

Volunteer closing simulation

Voter result-related flow

Use the interactive buttons, cards, or navigation controls to move through each step.

6. Allow location access if requested
The website may ask for location permission. If enabled, this can help provide more contextual election guidance.

Click Allow if you want location-based support.

Click Block if you do not want to share your location.

The platform should still remain usable even if location access is denied.

7. Exit or reset when needed
If you want to restart your flow, use the exit, back, or reset controls available in the interface.

This is useful when:

You selected the wrong role.

You want to try another user journey.

You want to repeat a simulation.

Suggested user flow
A simple way to use the platform is:

Open the website.

Read the introduction.

Choose your role.

Follow the guided steps.

Complete the interactive simulations.

Review the final stage or outcome shown.

Reset and try another role if needed.

Best experience tips
Use a laptop or desktop for smoother interaction.

Keep your internet connection stable.

Read each step before moving to the next one.

Use the simulations carefully to understand the real-world process.

Grant permissions only if you are comfortable doing so.

What users should expect
Users should expect:

A guided election learning experience.

Interactive role-based navigation.

Simple process explanation instead of complex legal language.

Simulated workflows for better understanding.

If something does not work
Try the following:

Refresh the page.

Restart the journey.

Reopen the website.

Check whether a popup or permission request is waiting for your response.

Try another browser if interactions are not responding correctly.

Intended purpose
This website is intended as an educational and guidance-based election assistant. It helps users understand the process better through role-specific, interactive learning.
